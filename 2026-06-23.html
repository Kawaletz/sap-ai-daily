Neue ABAP-Funktionen im Juni-Release: RAP-Draft-Erweiterungen, CDS-Signaturvergleich & ATC Clean Core für ECC UPDATE

  • RAP Draft – neue Feature-Control-Attribute: Im ABAP RESTful Application Programming Model (RAP) unterstützt Draft jetzt zwei neue Attribute in der Feature Control: read-only create und mandatory update. Das ermöglicht feingranularere Steuerung des Draft-Lebenszyklus in Fiori-Apps – wichtig für KI-generierte Formulare.
  • CDS-Migrationsassistenz – Signaturvergleich: Bei der Migration von DDIC-basierten Views auf CDS View Entities bieten die ABAP-Entwicklungstools jetzt einen Signaturvergleich an. Das reduziert Fehlerquellen bei diesem Schritt erheblich – ein direkter Vorteil für KI-gestützte Migrationsprojekte.
  • ATC Clean-Core-Analyse auch für ECC: Neu ist eine ATC-Variante, die Clean-Core-Analysen sowohl auf SAP ECC als auch SAP S/4HANA durchführt. On-Premise-Teams können den Custom-Code-Stand jetzt bereits vor einer S/4-Migration systematisch bewerten.
  • Analytics WYSIWYG-Editor – mehrere Datenprovider: Der WYSIWYG-Editor der Review-Booklets unterstützt jetzt mehrere Datenprovider pro Booklet, was komplexere Analysen in einer einzigen Ansicht ermöglicht.
