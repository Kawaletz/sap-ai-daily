ABAP Classic AI

S/4HANA Custom Code Migration Agent: Agentische Massenmigration von ECC nach S/4HANA

  • NEU Der nächste Schritt nach den AI-Skills: SAP stellt (15. Juni) den S/4HANA Custom Code Migration Agent vor – einen autonomen, kontextbewussten KI-Agenten innerhalb der SAP Joule for Developers, ABAP AI capabilities, der die typische Sequenz eines Custom-Code-Migrationsentwicklers weitgehend selbstständig abarbeitet. Statt vereinzelter AI-Quickfixes übernimmt der Agent jetzt die Massentransformation ganzer Custom-Code-Pakete von ECC nach SAP S/4HANA – mit Human-in-the-Loop nur dort, wo eine Entscheidung nötig ist.
  • So läuft der agentische Loop: Der Agent triggert die ATC-Analyse zur S/4HANA-Readiness, interpretiert die Ergebnisse, kategorisiert nach Komplexität, führt zunächst deterministische Quick Fixes als Massenoperation aus und wendet danach AI-basierte Fixes an. Für jeden AI-Fix wird ein Confidence Score berechnet – nur Fixes mit hohem Score und ohne Syntaxfehler werden automatisch übernommen; unsichere Vorschläge landen als auskommentierte Kommentare zur manuellen Prüfung.
  • Transparenz & Kontrolle: Alle Änderungen werden in einem einzigen dedizierten Transportauftrag protokolliert, alte Codezeilen bleiben auskommentiert und revertierbar, und die AI-Fixes inkl. Confidence Score werden als CSV für das Review exportiert. Der Agent läuft im Batch-Modus – Entwickler aktivieren am Ende nur die geänderten Objekte.
  • Technische Voraussetzungen: SAP S/4HANA-Zielsystem ab Release 2021 (Private Cloud/On-Prem, On-Prem-Verfügbarkeit unter speziellen RISE-Vertragsbedingungen), Entitlement für Joule for Developers ABAP AI, ADT for Eclipse Version 3.60 mit aktiviertem ADT MCP Server sowie ein AI-Assistant (GitHub Copilot oder Amazon Q). Custom Instructions/Rules werden gemäß SAP-Notes 3751537 (Copilot) bzw. 3751829 (Amazon Q) ergänzt.
