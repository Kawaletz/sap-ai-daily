# SAP AI Daily

**Tägliches KI-Briefing für die SAP-Welt**

A professional, modern static blog covering daily AI developments in the SAP ecosystem. Built with clean HTML, responsive design, and Google Fonts for a contemporary look.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Modern Styling**: Clean, professional aesthetic with a dark blue header and accent colors
- **Color-Coded Sections**: Each major section has its own color for easy visual navigation
  - Orange: ABAP & Classic SAP AI Focus
  - Blue: SAP Cloud & AI Platform News
  - Purple: Community & Social Media
  - Green: Learning & Certifications
  - Red: Video Resources & Events
- **Sticky Table of Contents**: Collapsible sidebar for easy navigation (desktop)
- **Typography**: Google Font "Inter" for modern, professional appearance
- **Comprehensive Content**: Full briefing with links, source citations, and curated community insights
