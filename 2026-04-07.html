Neu MCP Server für SAP ECC & S/4HANA: Unlimited ABAP Add-On

  • Ein neuer Community-Blog-Post beschreibt einen ABAP Add-On, der einen MCP Server direkt in SAP ERP-Systeme integriert – und das bereits ab ABAP Release 7.01. Das Besondere: keine separaten OData-Services nötig.
  • Kernfunktion: Tabellen, Views und CDS Views werden in einer ABAP-Ansicht registriert und vom MCP Server automatisch als Tools bereitgestellt. Create-, Update- und Delete-Aktionen erfordern nur je eine ABAP-Klasse.
  • Verbindung zum MCP Server kann direkt über einen SICF-Node oder via SAP Cloud Connector erfolgen – damit ist auch ein sicherer On-Premise-Zugang aus der Cloud möglich.
  • Die Lösung ist mit jedem Agentic SDK kompatibel und eignet sich besonders für Unternehmen, die klassische SAP ECC-Systeme AI-fähig machen wollen, ohne die Clean-Core-Architektur zu gefährden.

  • Ein neues GitHub-Projekt bringt den MCP Server direkt als Eclipse Plugin – damit läuft der Server innerhalb von Eclipse und gibt Claude Code und anderen AI-Tools vollen Zugang zur ABAP-Codebase: Suchen, Lesen, Schreiben, Testen und Aktivieren von ABAP-Objekten.

Neu Blog: Die Zukunft der Enterprise AI – MCP Server und Agent-to-Agent Communication

  • Ein neuer Blog-Post aus der SAP Community Bengaluru beleuchtet zwei komplementäre Protokolle, die die Enterprise-AI-Architektur 2026 prägen: MCP (Model Context Protocol) für die Kommunikation zwischen AI und Systemen, und A2A (Agent-to-Agent Protocol) für die Kommunikation zwischen verschiedenen AI-Agenten.
  • MCP löst das Problem der Integration: Ein universeller „Übersetzer", der den Bedarf an Custom-Integrations eliminiert. SAP unterstützt MCP seit TechEd 2025 in der gesamten BTP-Plattform, HANA Cloud erhält ab Q1 2026 vollen MCP-Support.
  • A2A geht einen Schritt weiter: Agents können sich gegenseitig entdecken, kommunizieren und koordinieren – was komplexe Multi-Step-Workflows erst ermöglicht. Einige Unternehmen berichten bereits von bis zu 50 % Kosteneinsparungen bei der AI-Infrastruktur durch reduzierte Integrationskomplexität.
  • Das Fazit des Posts: Die Zukunft der Enterprise AI wird durch die tiefe Integration von LLMs in bestehende Enterprise-Systeme getrieben – und MCP zusammen mit A2A liefert dafür die notwendige Standardisierungsgrundlage.
