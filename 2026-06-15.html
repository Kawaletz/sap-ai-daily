ADT for VS Code: Patch 1.0.1 bringt RFC-Passwort-Login und AGENTS.md-Template

  • UPDATE Schnelle Reaktion auf das Community-Feedback: Zwei Wochen nach dem GA-Release der ABAP Development Tools für VS Code veröffentlicht das ADT-Team (12. Juni) den Patch 1.0.1.   • Ausblick: Das nächste Release ist für den kommenden Monat geplant – mit weiteren Objekttypen (RAP Behavior Extensions, DDIC-Tabellen/-Strukturen, Programme/Includes, Funktionsgruppen) und neuen MCP-Tools (Unified-Diff-Tool für Transporte, ABAP-Test-Cockpit-Tools).

Re-invented ABAP AI ist da: SAP Joule for Developers jetzt als Service über den ABAP AI Hub

  • NEU Allgemeine Verfügbarkeit der neugestalteten ABAP-AI-Capabilities: SAP hat am 12. Juni die offiziell verfügbaren, neu konzipierten SAP Joule for Developers, ABAP AI capabilities – inklusive agentischer Fähigkeiten – bekanntgegeben. Sie werden jetzt als Service über den neuen ABAP AI Hub Service angeboten, der side-by-side neben den ABAP-Systemen läuft.
  • Release-unabhängig bis zurück zu 2021: Unterstützt werden SAP BTP ABAP Environment, SAP S/4HANA Cloud Public Edition sowie – neu erweitert – SAP S/4HANA Cloud Private Edition für Releases ab 2021. Statt an Release-Zyklen gekoppelt zu sein, konsumieren Kunden die KI-Services parallel zu ihren bestehenden Systemen – auch auf niedrigeren Releases ohne Upgrade.
  • Eine Lizenz, ein Onboarding: Mit einer einzigen verbrauchsbasierten Lizenz (AI Units, SKU 8019164 für Kunden) können die Capabilities über SAP for Me aktiviert und produktübergreifend genutzt werden. Authentifizierung läuft über Cloud Identity Services, KI-Requests werden im ABAP AI Hub Service auf SAP BTP gemessen.
  • Capability-Matrix: Predictive Code Completion, ABAP/CDS-Code-Erklärung, ABAP-Unit- und CDS-Test-Generierung sowie Documentation Chat sind auf allen Produkten verfügbar; OData-UI-Service-from-Scratch und RAP Business Logic Prediction nur ab PCE 2025/Public; der Custom Code Migration Assistant ist exklusiv für die Private Edition (2021–2025). Dazu kommen alle Fähigkeiten des gewählten Coding-Agenten via ADT MCP Server.
