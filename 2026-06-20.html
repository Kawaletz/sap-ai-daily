SAP Juni-Patchday 2026: Vier kritische Sicherheits-Fixes für NetWeaver ABAP

  • UPDATE 15 Security Notes, davon 4 kritisch: SAP hat zum Juni 2026 Security Patch Day insgesamt 15 Sicherheitshinweise veröffentlicht. Besonders dringend sind zwei Schwachstellen im ABAP-Umfeld: CVE-2026-44748 (CVSS 9,9) betrifft XML Signature Wrapping in der SAML-Authentifizierung für SAP NetWeaver AS ABAP und die ABAP Platform – ein authentifizierter Angreifer mit geringen Rechten kann manipulierte XML-Dokumente einschleusen und so Privilegien eskalieren. CVE-2026-27671 (CVSS 9,8) ist noch gefährlicher: Memory Corruption im RFC-Protokoll des AS ABAP Kernels – kein Login nötig, ein einzelnes speziell präpariertes RFC-Paket reicht.
  • Priorisierung: CVE-2026-44751 (CVSS 7,1) schließt eine fehlende Berechtigungsprüfung in SAP_BASIS 700–816. Empfohlen wird, SAP Note 3717897 (RFC Memory Corruption) und Note 3746332 (SAML XML Signature Wrapping) vorrangig einzuspielen. Für ABAP-Umgebungen mit KI-Integration (RFC-Aufrufe von externen Agenten, Joule) ist rasches Patchen besonders wichtig.
